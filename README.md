# Fixed-Wing UAV CAD Design

## Project Overview

This project presents a fixed-wing unmanned aerial vehicle (UAV) concept developed as a complete exterior CAD model. The design integrates the fuselage, wing, empennage, propeller, control surfaces, and landing gear into a single aircraft configuration.

The model was created to support early-stage aircraft layout studies, geometric proportioning, component placement, and technical visualization.

## Design Objectives

- Develop a complete fixed-wing UAV configuration
- Establish suitable geometric proportions between the fuselage, wing, and tail
- Integrate the propulsion system, control surfaces, and landing gear
- Create a clean exterior CAD model for later aerodynamic and structural studies
- Prepare a concept that can be refined for simulation, prototyping, and performance analysis

## Main Design Features

- Conventional fixed-wing configuration
- Tractor-propeller arrangement
- Straight wing with integrated control surfaces
- Conventional tail configuration
- Tricycle landing-gear arrangement
- Streamlined fuselage geometry
- Fully assembled exterior model

## Current Status

The project is currently at the concept and CAD-development stage. The exterior geometry and major aircraft components have been integrated. Further work may include aerodynamic analysis, mass estimation, stability calculations, structural refinement, and prototype development.

## Limitations

- The current model is an early-stage concept
- Detailed aerodynamic validation has not yet been completed
- Structural sizing and material selection are not included
- Internal systems and manufacturing details require further development
- Flight performance values should not be inferred from the CAD model alone
